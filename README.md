# Enhancing Accuracy of Electricity Price Forecasting

> **Author:** Pavani Yelisetti  
> **Data Source:** ERCOT (2024) — Load Zone North  
> **Models:** Naïve · ARX · NARX · Holt-Winters · Forecast Combinations

---

## Overview

Electricity prices are notoriously volatile, driven by fluctuating demand, weather, fuel costs, and grid conditions. This project builds and compares four forecasting models on hourly ERCOT price data, then evaluates ensemble combination methods to further reduce error.

**Key finding:** A weighted average combination of all four models achieves the lowest MAE of **13.15 EUR/MWh**, outperforming every individual model.

## Dataset

| Field | Description |
|---|---|
| Source | ERCOT (Electric Reliability Council of Texas), 2024 |
| Zone | Load Zone North |
| Features | `date`, `hour`, `price (EUR/MWh)`, `load`, `day_of_week` |
| Lag features | Prices at –24h, –48h, –168h (daily & weekly cycles) |
| Training set | 351 days |
| Test set | Last 7 days (168 hours) |

---

## Models

### 1. Naïve
Uses the price from the same hour one week ago as the forecast. Surprisingly competitive due to strong weekly seasonality in electricity prices.

### 2. ARX — AutoRegressive with Exogenous Variables
Multiple linear regression on lagged prices (24h, 48h, 168h) and load. Tested in two configurations:
- **Rolling window** — retrains on a sliding history (adapts to recent trends)
- **Fixed window** — trains once on the full 351-day history (more stable)

### 3. NARX — Neural ARX
A neural network that uses the same inputs as ARX but learns nonlinear relationships between price lags, load, and future price. Best single-model performer.

### 4. Holt-Winters (Triple Exponential Smoothing)
Decomposes the series into level, trend, and seasonal components. Does **not** use load as an input, which limits its accuracy during demand spikes.

## Forecast Combination Methods

| Method | Weights | MAE |
|---|---|---|
| Simple Average | Equal (0.25 each) | 13.56 |
| Trimmed Mean | Drops HW, equal on rest | 13.40 |
| **Weighted Average** | Naïve 0.35 · NARX 0.30 · ARX 0.25 · HW 0.10 | **13.15** |

## Results Summary

| Model | MAE | Notes |
|---|---|---|
| Naïve | 13.27 | Strong baseline; leverages weekly seasonality |
| ARX (rolling) | 14.82 | Adapts to recent trends but accumulates forecast error |
| ARX (fixed) | 11.20 | Best single ARX; stable training window |
| NARX | 14.44 | Captures nonlinear patterns |
| Holt-Winters | 17.43 | No exogenous load variable; underperforms at spikes |
| Simple Avg | 13.56 | Better than ARX, NARX, HW individually |
| Trimmed Mean | 13.40 | Drops HW; slight improvement |
| **Weighted Avg** | **13.15** | **Best overall** |

## Quickstart

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Add your data

Place the ERCOT hourly CSV in `data/ercot_north_2024.csv` with columns:

```
datetime, price, load, day_of_week
```

### 3. Run the full pipeline

```bash
python src/evaluate.py --data data/ercot_north_2024.csv
```

### 4. Or run the notebook interactively

```bash
jupyter notebook notebooks/price_forecasting.ipynb
```

## Key Conclusions

- **Rolling window** adapts to recent trends — best for highly dynamic markets
- **NARX** is the best single model, capturing nonlinear price–load relationships
- **Naïve** outperforms several complex models thanks to strong weekly seasonality
- **Weighted average combinations** consistently beat individual models — model diversity is the main driver
- **Holt-Winters** underperforms without access to exogenous load data; should be down-weighted or excluded
